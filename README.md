# Wechat-Robot
一个玩微信的机器人
基于wcferry的robot demo魔改，增加了一些运维实用功能
感谢大佬lich0821开源的wcferry及其机器人项目

## 目前功能
- AI助手
- 当前gpt-key密钥查询
- 查ip归属地
- ip测速
- 查域名ip（类似nslookup）
- 全网点歌台（暂未实现xml发送）
